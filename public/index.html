Sobre

Desenvolvo aplicações full-stack utilizando JavaScript e C#, tanto no front-end quanto no back-end, aplicando princípios de orientação a objetos. No front-end, utilizo React para JavaScript, com Tailwind ou Sass para estilização, e Blazor para criar interfaces dinâmicas com C#. No back-end, trabalho com Node.js e C#, adotando arquiteturas como serverless e microserviços conforme a necessidade. A autenticação é implementada com JWT e OAuth em ambas as linguagens, garantindo segurança e controle de acesso. Também foco na qualidade do código, implementando testes automatizados para garantir soluções robustas e confiáveis. Meu objetivo é entregar aplicações seguras, eficientes e escaláveis, utilizando as melhores práticas e tecnologias disponíveis.
